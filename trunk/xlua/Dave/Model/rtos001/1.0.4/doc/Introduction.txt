\page RTOS001_intropage Introduction
The CMSIS-RTOS API is a generic RTOS interface for Cortex-M processor-based devices. 
CMSIS-RTOS provides a standardized API for software components and gives therefore serious benefits to the users and 
the software industry.<BR>
Please refer [1] for details.<BR>

CMSIS-RTOS API app provides GUI base configuration for CMSIS-RTOS API Library provided by ARM. 
The CMSIS-RTOS implementation of this RTOS001 App is based on ARM RTX and is developed by ARM and is published under the popular 
open-source BSD license and is available for download from http://www.keil.com/demo/eval/rtx.htm.
The official documentation of the used CMSIS RTX version can be downloaded separately from 
http://media.infineon.com/DAVE_3/CMSIS_RTX/CMSIS_RTX_Doc.zip


\section RTOS001_scopesec Scope
Scope of this App is to provide an RTOS implementation (source code) using the CMSIS-RTOS APIs and Configuration for CMSIS-RTOS API layer.

\section RTOS001_abbreviation
<OL>
<LI>OS : Operating System</LI>
<LI>RTOS: Real Time Operating System</LI>
<LI>RTX: Real Time Executive</LI>
<LI>CMSIS: Cortex Microcontroller Software Interface Standard</LI>
</OL>

\section RTOS001_reference
cmsis documentation can be downloaded from :
<BR>
[1] http://www.onarm.com/Cmsis/
<BR>
[2] To view updated version of the CMSIS RTOS API documentation <a href="..\cmsis\html\index.html"><B>Click here</B></a>

\section RTOS001_limitation
[1] This port of RTX does not start main as a thread.
<BR>
[2] Starting the Kernel shall be triggered from user code by explicitly calling the 'osKernelStart()' API.
@}
