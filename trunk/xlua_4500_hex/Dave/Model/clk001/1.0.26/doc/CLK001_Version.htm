1.0.22 01-Nov-12 1. The following changes are made in system clock configuration page:
   a. Removed External Digital Clock and External Oscillator from System Clock   Source selection. Equivalent functionalaity now available in Prescaler Mode under PLL Mode.
   b. Added Back Up Clock  as PLL Input source.
   c. Added selection of PLL Mode (Normal/Prescaler). This functionality was previously under  External Digital Clock and External Oscillator from System Clock Source selection.
   d. Added two additional options for Back Up Clock trimming. The default trimming option is now 'Automatic Trimming using Internal Slow Clock Source'
   e. K1 divider is now editable in PrescalerMode
2. The following changes are made in the System Clock Initialization sequence:
  a. Backup Clock Trimming is done before Configuring the PLL. The trimming code contains code for additional trimming options newly added.
  b. Clock enable for different peripherals are now split in to separate functions.
  c. USB Clock initialization is now in a different function.
3. The following changes are made in USB clock configuration page:
  a. Added UI for displaying calculated NDIV and PDIV values.
  b. The target VCO frequency for USB PLL is now changed to 384 MHz. Based on this PDIV and NDIV calculations are now different.
  c. The default value of USBDIV is now 4. This means 48 MHz is generated from 384 MHz VCO frequency.
4. In the EBU clock configuration page the Actual EBU frequency is now not editable.
5. In the WDT clock UI element added to show the actual WDT clock frequency and for selecting standby clock source.
6. The code for configuring Ultra Low Power oscillator has been changed.
1.0.26 05-Dec-2012 1. XMC42xx support added.
