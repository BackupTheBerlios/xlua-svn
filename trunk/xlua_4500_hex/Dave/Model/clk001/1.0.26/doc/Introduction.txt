\page CLK001_intropage Introduction 

<img src="..\CCU.PNG" title="Clock Control Unit Block Diagram"  />

CLK001 App provides UI option to configure the System Clock Source and option to enable/gate  the Peripheral module clocks in the core domain.
The above block diagram is of the Clock Control Unit. The App allows the System frequency(Fsys) to be configured from one of the following sources.
<UL>
<LI>
PLL
</LI>
<LI>
Internal Oscillator (OFI)
</LI>
<LI>
External Oscillator ( Prescalar mode of PLL)
</LI>
<LI>
External Clock Source( Prescalar mode of PLL)
</LI>
</UL>
<UL>

<LI>
<b>PLL:</b>
</LI>
The main PLL converts a low-frequency external clock signal to a high-speed internal clock.It allows the use of input and output frequencies of a wide range by varying the different divider factors(N,P,K2,K1).
The PLL consists of a Voltage Controlled Oscillator (VCO) with a feedback path. A divider in the feedback path (N-Divider) divides the VCO frequency down. The resulting
frequency is then compared with the externally provided and divided frequency (PDivider).The output is finally divided by a factor(K2 Divider).
<img src="..\PLL.PNG" title="PLL Block Diagram"  />

The input clock for the PLL   can be one of the following clock sources:
<UL>
<LI>
 Internal Oscillator
</LI>
<LI>
 External Oscillator
</LI>
<LI>
 External Clock Input
</LI>
</UL>

<LI>
<b>Internal Oscillator (OFI):</b>
</LI>
The Internal Oscillator is the default clock after start-up. It is used for bypassing the PLL for startup of the system without external clock. Furthermore it can
be used as independent clock source for the watchdog module or even as system clock source during normal operation. While in prescaler mode this clock is automatically used
as emergency clock if the external clock failure is detected.
<LI>
<b>External Oscillator ( Prescalar mode of PLL):</b>
</LI>
An external Oscillator load circuitry can be connected to pins, XTAL1 and XTAL2. The exact values and related operating range depend on the crystal used.User can configure
the crystal frequency in the App UI. 
<LI>
<b>External Clock Source( Prescalar mode of PLL):</b>
</LI>
In this usage an external clock signal is supplied directly not using an external crystal and bypassing the amplifier of the oscillator. The input frequency must be in the range
from 4 to 40 MHz.When using an external clock signal it must be connected to XTAL1.XTAL2 is left open i.e. unconnected.
</UL>

 Desired frequency limited to combo box range.

</LI>
