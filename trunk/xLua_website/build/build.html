Download and compile xLua

Requirements

  1. Install latest Dave™3 link
  2. You the following choices to download a project:
    1. Download one of configured project directly from download files.
    2. Import project directly from Berlios SVN. Remember that you can do this from Dave™3 by installing a subversion

Configuration

The project in the file repository and the Lua in the trunk of svn are already working and there is no need to configure them in the most of cases. However it is possible to customize the project by changing the following settings of

  1. RTOS (not covered here see Dave™ RTOS001 app documentation)
  2. xlua_settings
  3. Adding more apps and/or C libraries and interfacing them using SWIG

Compile & Download

  1. Press Generate Code
  2. Due to a bug of Dave™3 after code generation the project forgets the include paths. It is required to them to the project following the bewlo method:
    1. Select Properties-> C/C++ General -> Paths & Symbol
    2. Press Inport settings
    3. Select File :<todo>
  3. Compile it
  4. Download and run.
